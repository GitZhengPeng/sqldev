# ZiWei UI V3 视觉规范（专业版）

## 1. 视觉目标
- 形成单一阅读路径：输入 → 排盘 → 白话摘要 → 专业细解。
- 强化方盘主视图，降低次要模块干扰。
- 用稳定色彩语义建立“可扫读”的专业界面。

## 2. 颜色语义
- 主轴金色：宫位标题、关键信息、命理主标签。
- 主星（重点）朱红：主星名与关键星曜。
- 辅星（信息）青蓝：辅助判断信息。
- 四化固定色：
  - 禄：绿色
  - 权：橙色
  - 科：蓝色
  - 忌：红色
- 风险类提示统一红色系，建议类统一蓝青系。

## 3. 信息层级
- L1（主结论）：白话结论、当前大限、当前流年落宫。
- L2（证据层）：宫位主星、三方四正、四化落点。
- L3（展开层）：专业解释、操作建议、风险细项。

## 4. 组件规范
- 输入面板：
  - 基础参数默认展开。
  - 高级参数（校时/起法）默认折叠，提供摘要行。
- 方盘面板：
  - 维持最高对比度与视觉权重。
  - 宫位内部保持三段结构：标题/星曜/运限。
- 摘要卡片：
  - 首屏先给“白话结论”。
  - 二屏给“依据 + 专业解读 + 建议 + 风险”。

## 5. 交互规范
- 点击摘要项高亮关联宫位。
- 点击宫位自动聚焦对应解读段（后续阶段可加）。
- 排盘成功后提供可解释提示（含校时修正、规则口径）。

## 6. 响应式
- Desktop：方盘为主，摘要侧栏并列。
- Tablet：方盘与摘要上下排列，保持方盘优先。
- Mobile：保留白话摘要默认展开，专业内容按节折叠。

## 7. 验收标准
- 新用户 5 秒内可定位“排盘按钮 + 当前主结论”。
- 同一命盘在深浅主题下可读性一致（对比度达标）。
- 高级参数折叠后，首屏信息输入区域高度下降明显。
