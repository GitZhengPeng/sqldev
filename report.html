Security

安全问题 ×6

覆盖 CSP 策略、CDN 完整性、剪贴板安全、内存泄漏、输入限制、网络连接等方面。

SEC-01 已修复

缺少 Content-Security-Policy

原始文件无 CSP meta 标签，页面容易受到 XSS 注入攻击。

+ <meta http-equiv="Content-Security-Policy" content="default-src 'self'; script-src 'self' 'unsafe-inline' 'unsafe-eval' https://unpkg.com; ...">
SEC-02 已修复

CDN 资源无完整性校验

Vue.js 从 unpkg 加载时缺少 crossorigin 属性，若 CDN 被劫持可注入恶意代码。

+ crossorigin="anonymous"
SEC-03 已修复

剪贴板回退实现不安全

document.execCommand('copy') 回退方案中，临时 textarea 未设置为屏幕外，用户可能看到闪烁。

+ ta.style.cssText = 'position:fixed;left:-9999px;top:-9999px'
SEC-04 已修复

URL.revokeObjectURL 时序错误

在 DOM 元素移除后立即调用 revokeObjectURL，下载可能尚未开始就被撤销，导致下载失败。

+ setTimeout(() => URL.revokeObjectURL(objUrl), 1000)
SEC-05 已修复

缺少输入大小限制（ReDoS 风险）

超大 DDL 输入可能导致正则解析引擎卡死，造成页面无响应。

+ if (input.length > 500000) return '-- 输入内容过大（超过 500KB），请拆分后转换'
SEC-06 已修复

缺少 preconnect 预连接

未对 Google Fonts 和 unpkg CDN 设置 DNS 预解析，增加了 MITM 攻击窗口并拖慢加载。

+ <link rel="preconnect" href="https://fonts.googleapis.com"/>
+ <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin/>
+ <link rel="preconnect" href="https://unpkg.com"/>
Performance

性能问题 ×4

涵盖网络预连接、浏览器兼容性、CSS 渲染优化和 GPU 合成层管理。

PERF-01 ✓ 已修复

缺少 preconnect 资源提示

Google Fonts 和 unpkg 域名的 DNS 解析、TCP 握手、TLS 协商延迟叠加，影响首屏渲染。

修复：添加 3 个 preconnect link 标签
PERF-02 ✓ 已修复

backdrop-filter 无 WebKit 前缀

Safari 浏览器不识别无前缀的 backdrop-filter，导致毛玻璃效果失效。

修复：补充 -webkit-backdrop-filter
PERF-03 ✓ 已修复

缺少 CSS Containment

.app 容器未使用 CSS containment，子元素布局变化会触发全页重排。

修复：添加 contain: layout style
PERF-04 ✓ 已修复

伪元素缺少 GPU 合成提示

body::before 的大面积渐变层未标记 will-change，可能反复触发重绘。

修复：添加 will-change: auto

// Before:
if (v === 'TRUE') ...
// After:
if (v === 'TRUE' || v === "'1'") ...
